## 评语生成规则

年报底部的个性化评语根据你在社区中的百分位排名自动生成：

| 条件 | 评语 |
|------|------|
| Stars 百分位 ≥90% | 你是社区中的 Star 收割机 |
| Stars 百分位 ≥70% | 你的项目很受欢迎 |
| 贡献百分位 ≥90% | 代码贡献量名列前茅 |
| 贡献百分位 ≥50% | 持续为社区贡献代码 |
| 研究方向 ≥5 个 | 涉猎 N 个研究方向，堪称全能选手 |
| 研究方向 ≥3 个 | 在 N 个方向都有建树 |
| 深夜提交 ≥50% | 是一位资深夜猫子开发者 🦉 |
| 关注者百分位 ≥80% | 在社区中颇具影响力 |

百分位计算方式：统计所有成员中该指标比你高的人数作为排名，`百分位 = (1 - 排名/总人数) × 100`。符合多条规则时会拼接显示，均不符合则显示"继续加油，你的开源之旅才刚刚开始！"
